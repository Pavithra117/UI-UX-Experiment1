# UI-UX-Experiment1

## Aim:

To study the fundamentals of User Interface (UI) and User Experience (UX), understand their differences through real-world examples, analyze their relationship in design, and explore commonly used UI/UX design tools.

## Algorithm:

1.Study the basics of UI and UX.

2.Compare UI and UX using real-world applications.

3.Create a simple wireframe and analyze user experience.

4.Explore UI/UX tools like Figma, Adobe XD, and Sketch.

5.Develop a basic prototype and evaluate its usability.

[ui_ux .pdf](https://github.com/user-attachments/files/28543638/ui_ux.pdf)

## Result:

The fundamentals of UI and UX were understood, their differences and relationship were analyzed, and various UI/UX design tools were successfully explored and demonstrated.
